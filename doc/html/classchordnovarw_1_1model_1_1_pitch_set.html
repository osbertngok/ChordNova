◆ get_tension()

double chordnovarw::model::PitchSet::get_tension ( ) const
nodiscardoverridevirtual

Calculate the tension value.

Traverse all intervals between the notes For each interval, calculate the tension score for second note in the note pair that are high enough.

Per https://www.bilibili.com/video/BV1Ja411s7mA, the 5 problems above are partially handled:

  • Tension value is not normalized by note count: NOT addressed. Its value is proportional to \(n^2\).
  • The weight vector is arbitrary: PARTIALLY addressed. Weight vector used here is proposed by 赵晓生, based on consonance of an interval.
  • The arrangement of the chord should impact tension score: NOT addressed in this algo. This algo however does consider penalty when notes are in lower register.
  • The prime form of an asymmetrical forte arrangement should have different score of the inverse form: NOT addressed in this algo. The weight vector is symmetrical.
  • Timbre also impacts tension value. NOT addressed.
Returns
tension value
