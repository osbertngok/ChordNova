◆ single_chroma

vector<int> chordnova::chord::ChordData::single_chroma

The chroma value corresponding to each note, in the order of Circle of Fifths.

It does not distinguish spelling difference of two enharmonically equivalent pitch classes, like C# and Db.

For example, converting Csus2 (C4 D4 G4) to the positions of Circle of Fifths would use 0, 2, 1. This single_chroma variable would always store the values in monotonically increasing order, so the final result is [0, 1, 2].

Calculating this would help to calculate the minimal span on the Circle of Fifths.
